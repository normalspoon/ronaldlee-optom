# Ronald Lee Optometrist

Static website for Ronald Lee Optometrist.

## Files

- `index.html` contains the page content and structure.
- `styles.css` contains the site styling and responsive layout.
- `script.js` contains the small front-end interactions.
- `assets/` contains site images.
- `CNAME` keeps the custom domain mapped correctly.

## Local Preview

Run a simple local server from the project root:

```sh
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Deployment

This site is deployed through the current GitHub/Hostinger setup.

- Push changes to `main` to publish updates.
- Keep the `CNAME` file in place for the custom domain.
- Do not use Netlify for deployment on this project.
